# api-basic

# REST API
List of user routes:

| Route                 | HTTP    | Description         |  
| --------------------- | ------- | ------------------- |
| /api/users            | GET     | Get all the users   |
| /api/users/:id        | GET     | Get a single user   |
| /api/users            | POST    | Create user         |
| /api/users/:id        | PUT     | Update user         |
| /api/users/:id        | DELETE  | Delete user         |

## Usage
with only npm:

```

$ npm install
$ npm start

```
